# Dropdown

Um menu dropdown.

## Quando Utilizar

Quando um conteúdo ou menu precisa ser exibido em uma janela dropdown.

## Exemplos

### Uso Básico

```jsx
import Dropdown from "@alboom/boom-components/data-display/dropdown"

const MyComponent = () => {
  const dropdownContent = (
    <div>
      Hi, I'm hidden until you interact with the dropdown.
    </div>
  )
  return (
    <Dropdown overlay={dropdownContent}>
      <a>Hover me!</a>
    </Dropdown>
  )
}
```

### Alinhamento

O Dropdown possui um alinhamento inteligente, alterado o alinhamento caso não haja espaço disponivel na tela para exibição do mesmo (ex: um dropdown abre para a parte superior caso o elemento
de disparo esteja posicionado no final da tela). Você pode, entretanto, controlar o alinhamento horizontal em relação ao componente de disparo com a propriedade align:

```jsx
import Dropdown from "@alboom/boom-components/data-display/dropdown"

class MyComponent extends React.Component {
  render(){
    return (
      <div>
        <Dropdown align="left" overlay={<div>Oi, estou alinhado a esquerda</div>}><a>Alinhado a esquerda</a></Dropdown>
        <Dropdown align="right" overlay={<div>Oi, estou alinhado a direita</div>}><a>Alinhado a direita</a></Dropdown>
        <Dropdown align="center" overlay={<div>Oi, estou alinhado ao centro</div>}><a>Alinhado ao centro</a></Dropdown>
      </div>
    )
  }
}
```

### Dropdowns controláveis

É possivel controlar a visibilidade de um Dropdown a partir de um componente pai, tornando-o controlado. Exemplo:

```jsx
import Dropdown from "@alboom/boom-components/data-display/dropdown"
import Button from "@alboom/boom-components/general/button"
import Icon from "@alboom/boom-components/general/icon"

class MyComponent extends React.Component {
    constructor(props){
      super(props)
      this.state = {
        visible: false
      }
    }

    render(){
      const { visible } = this.state;
      const content = (<div>
        <ul>
          <li key="some key 1">Item 1</li>
          <li key="some key 2">Item 2</li>
        </ul>
      </div>);

      return (
        <div style={{display: "flex", alignItems: "center", height: "40px"}}>
          <div>
            <Button onClick={()=>{this.setState({visible: true})}}>Open Menu</Button>
          </div>
          <div style={{height: "35px"}}>
            <Dropdown
              overlay={content} 
              visible={visible} 
              onVisibleChange={(visible) => {
                this.setState({visible: visible})
              }} 
              trigger="click"
            >
              <Button>
                <Icon kind="bold" group="arrows-diagrams" category="arrows" file="arrow-down-1.svg" size="16" color="#000000" />
              </Button>
            </Dropdown>
          </div>
        </div>
      )
    }
  }
```
