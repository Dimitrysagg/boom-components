# Button

Botões ajudam a tornar ações mais óbvias e ajudam usuários as executarem mais facilmente. Botões geralmente são compostos de texto e algumas vezes icones para comunicar a ação que acontecerá quando o usuário clicar no mesmo.

## Quando utilizar

Um botão significa uma operação (ou uma série delas). Clicar em um botão deve executar lógica de negócio, assim, utilize botão toda vez que necessitar executar lógica após interação do usuário
